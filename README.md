## 碎碎念

ChatGPT 于 2022 年底横空出世，开启了 **AIGC** 时代

而笔者的大部分课程开设于前 AIGC 时代，因此仓库的资料全部来自人工 Ctrl CV 和手写

毕业多年后回看，给人一种稚嫩的感觉，不过这也是本科生的正常水平

如果让高级 AI 模型来写，质量当然会很好，但也脱离了一般本科生水平
